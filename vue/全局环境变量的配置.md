# vue中全局环境变量的配置

## .env

在项目的根路径下创建一个文件 .env

```js
VUE_APP_URL = https://env.pai.com
```

在模板文件中使用

```js
data () {
    return {
        url: process.env.VUE_APP_URL
    }
}
```

## .env.development

开发环境, 如果有这个就走这个，

```js
VUE_APP_URL = https://env.pai.com
```

## env.production

生产环境下使用

```js
VUE_APP_URL = https://env.pai.com
```
